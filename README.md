# Sepomex API WordPress Plugin

A WordPress plugin for integrating with the Sepomex API to handle Mexican postal codes and addresses.

## Features

- Basic plugin structure
- Internationalization support
- Asset management (CSS/JS)
- Activation/Deactivation hooks

## Installation

1. Upload the `sepomex-api` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure the plugin settings as needed

## Usage

Add your custom functionality in the main plugin file or create additional files in the `includes` directory.

## Development

- Main plugin file: `sepomex-api.php`
- Assets: `assets/` directory
- Additional includes: `includes/` directory
- Translations: `languages/` directory
